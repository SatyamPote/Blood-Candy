## 🍬 Overview

Blood Candy Elite is a custom-built, native terminal-based command center designed for absolute workstation dominance. It bypasses traditional AI boundaries, granting full, uncensored, and unrestricted access to the host system.

## 🚀 Elite Features
- **Native TUI Dashboard**: High-speed terminal interface with crimson-dotted hacker aesthetics.
- **Danger-Full-Access**: Hardcoded unrestricted permissions for total system control.
- **Zero-Baggage Ignition**: Automatic session purging on every run for a clean slate.
- **Slash Command Suite**: Integrated control via `/run`, `/fresh`, `/status`, and more.
- **Graphify Integration**: Persistent knowledge management via [Graphify.net](https://graphify.net/).

## 🛠️ Usage
```powershell
npm run blood
```
Then ignite the engine:
```
BLOOD-CANDY $ /run
```
